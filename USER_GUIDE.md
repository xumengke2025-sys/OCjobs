# OC 一键求职新玩法 - 用户操作手册

欢迎使用 **OC 一键求职新玩法**！这是一个结合了 **AI 角色扮演** 与 **求职模拟** 的创新应用。

你可以上传任意小说角色、游戏人物或原创 OC（Original Character）的设定文档，系统将自动分析其性格、经历与技能，为其生成一份专业的求职简历，并自动模拟向多家公司投递简历的过程。你甚至可以扮演该角色，与性格各异的 AI 面试官进行即时对话。

---

## 核心功能

### 1. 🎭 角色简历生成
*   **一键解析**：支持上传 PDF/TXT/MD 格式的角色设定文件（如小说大纲、人物小传）。
*   **智能转换**：AI 会自动将角色的武功、魔法或特殊经历转化为现代职场技能（例如：将“千里传音”转化为“即时通讯架构能力”）。
*   **多风格支持**：自动识别角色语调（如武侠风、现代风），简历措辞与排版风格随之变化。

### 2. 🚀 自动投递系统
*   **批量投递**：简历生成后，系统会自动筛选并投递 5 家与角色能力匹配的公司。
*   **多样化雇主**：
    *   **现代角色**：投递给科技大厂、创业公司、传统国企等。
    *   **武侠角色**：投递给镖局、情报组织、朝廷卫队等（具备沉浸式世界观）。

### 3. 💬 沉浸式沟通
*   **性格化 HR**：不同的公司配备了不同风格的 HR（热情创业者、高冷大厂 HR、神秘接头人等）。
*   **角色化回复**：系统会根据你的角色设定，自动生成符合人设的回复建议（如侠客自称“在下”，职场人称“鄙人”）。

## 快速开始

### 环境准备
确保您的电脑已安装 Node.js 和 Python 环境。

1.  **启动项目**：
    在项目根目录下运行终端命令：
    ```bash
    npm run dev
    ```
    等待浏览器自动打开（默认地址：`http://localhost:5173`）。

---

## 操作指南（保姆级教程）

### 第一步：进入 OC 求职板块
项目启动后，点击顶部导航栏的 **"OC 求职"** 或直接访问 `/oc` 页面。

### 第二步：导入角色设定
在左侧【简历生成】面板中：
1.  **上传文件**：点击虚线框区域，上传你的角色设定文件（推荐 `.txt` 或 `.md`）。
2.  **或者使用样例**：点击下方的 **"一键体验样例"**（如：叶孤城），快速加载预设数据。
3.  **填写意向**（可选）：输入你想让角色应聘的岗位（如：`首席安全官`）和级别（如：`P8`）。

### 第三步：生成简历
1.  点击底部的 **"开始生成简历"** 按钮。
2.  系统将开始分析文档，大约需要 10-30 秒。
3.  **多角色选择**：如果你的文档中包含多个角色，系统会弹窗提示你选择其中一个进行生成。

### 4. 查看简历与投递记录
生成完成后，页面会有以下变化：
*   **右侧**：默认展示生成的 **在线简历预览**。你可以检查 AI 如何将“内功心法”包装成“核心抗压能力”。
*   **左侧**：自动切换到 **"求职中心"** 列表，显示已自动投递的 5 家公司。

### 5. 开始聊天
1.  在左侧【投递记录】列表中，点击任意一个公司（状态可能是“新招呼”、“已投递”等）。
2.  **切换视图**：右侧主视图会自动切换到 **"沟通消息"** 界面。
3.  **发送消息**：
    *   你可以直接在输入框输入内容。
    *   **推荐**：使用输入框上方的 **快捷工具**（如“📄 发送简历”、“📱 交换微信”）。
    *   **自动回复**：输入框中会预置一句符合你角色人设的开场白，直接回车即可发送。
4.  **体验反差**：
    *   尝试用武侠角色的语气去面试一家现代互联网公司，看看 HR 会如何反应（可能是一脸懵圈，也可能是陪你演戏）。

## 常见玩法推荐

*   **武侠穿越流**：上传一个古代大侠的设定，让他去面试现代的“安保总监”或“物流主管”。
*   **魔法转职流**：上传一个魔法师设定，让他去面试“特效师”或“能源工程师”。
*   **反派洗白流**：上传一个反派角色的设定，看看 AI 如何把他的“统治世界计划”包装成“宏大的市场占有率愿景”。
